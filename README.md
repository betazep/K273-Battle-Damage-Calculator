# K273-BDC Project

Static GitHub Pages site for Total Battle Kingdom 273 with a damage calculator.

## Pages
- `docs/index.html`: Landing page (links to the calculator).
- `docs/calculator.html`: Damage calculator (base costs, flat/tiered losses, summary).

## Structure
```
docs/
  index.html
  calculator.html
  css/styles.css
  js/main.js
```

## Usage
- Open `docs/index.html` locally or via GitHub Pages; click through to the calculator.
- Edit `docs/js/main.js` to adjust base cost defaults.
