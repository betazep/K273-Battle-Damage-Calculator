<div class="book-section">
    <h4>Comment construire votre coffre</h4>
    <ul>
        <li>Terminez les quêtes quotidiennes.</li>
        <li>Invoquez votre coffre dans l'onglet Événements.</li>
        <li>Appelez votre raid.</li>
        <li>Éjectez les joueurs qui ne suivent pas les consignes ci-dessous.</li>
        <li>Raids complets de 5 joueurs seulement, sinon annulez.</li>
    </ul>
</div>
<div class="book-section">
    <h4>Stratégie de raid et couleurs du coffre</h4>
    <ul>
        <li>Vert : minimum G2.</li>
        <li>Bleu : minimum G3.</li>
        <li>Violet : minimum G4.</li>
        <li>Or : minimum G5.</li>
        <li>Rouge : minimum G6.</li>
        <li>Argent : minimum G7.</li>
    </ul>
</div>
<p>Ne rejoignez pas un raid si vous êtes en dessous du niveau minimum de Guardsmen. Si d'autres rejoignent en dessous, retirez-les.</p>
