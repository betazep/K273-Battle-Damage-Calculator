#### Doomsday

**Sacrifiez votre armée pour gagner XP et Valor (vos troupes mourront).**

- Activez les bonus à l'avance.
- Si vous ne pouvez pas envoyer de grandes armées, utilisez uniquement les troupes les plus fortes.
- Si vous pouvez construire de grandes armées, empilez les troupes.
- Priorité à Carter.
- Utilisez Valor pour rechercher Archeology et Guardsmen.

**Gardez vos mercenaries (chasseurs de monstres épiques) pour ces événements.**
