<div class="book-section">
    <h4>Téléportation de ville</h4>
    <p>Comment téléporter votre ville :</p>
    <ul>
        <li>Trouvez un emplacement.</li>
        <li>Appuyez sur la case où vous voulez téléporter.</li>
        <li>Sélectionnez "city teleport".</li>
        <li>Si vous n'avez pas de téléport, vous devrez l'acheter avec de l'or.</li>
    </ul>
</div>
<div class="book-section">
    <h4>Où téléporter ?</h4>
    <ul>
        <li>Pour les grandes armées, vous avez besoin d'une case avec bonus "food production". Les armées ont besoin de nourriture. Plus la production est élevée, plus l'armée est grande.</li>
        <li>Pour les villes basées sur l'argent, le désert est le meilleur endroit.</li>
    </ul>
</div>
<p><strong>Important :</strong> Gardez votre ville aussi proche que possible du "groupe" du clan. Cela facilite les renforts, les caravanes, etc.</p>
