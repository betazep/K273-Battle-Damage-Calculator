#### Téléportation de ville

Comment téléporter votre ville :

- Trouvez un emplacement.
- Appuyez sur la case où vous voulez téléporter.
- Sélectionnez "city teleport".
- Si vous n'avez pas de téléport, vous devrez l'acheter avec de l'or.

#### Où téléporter ?

- Pour les grandes armées, vous avez besoin d'une case avec bonus "food production". Les armées ont besoin de nourriture. Plus la production est élevée, plus l'armée est grande.
- Pour les villes basées sur l'argent, le désert est le meilleur endroit.

**Important :** Gardez votre ville aussi proche que possible du "groupe" du clan. Cela facilite les renforts, les caravanes, etc.
