Si vous utilisez des catapultes d'un niveau supérieur à celui indiqué, seule la moitié de la quantité est nécessaire.

#### Citadelles elfiques

- **Niveau 10** : 100 catapultes niveau 2. Uniquement distance + Gargouilles.
- **Niveau 15** : 320 catapultes niveau 3. Uniquement montés + Gargouilles, Sangliers et Phénix.
- **Niveau 20** : 800 catapultes niveau 4. Uniquement mêlée + Griffons.
- **Niveau 25** : 3200 catapultes niveau 5. Uniquement mêlée + Griffons.
- **Niveau 30** : Boop & pray, catapultes niveau 6. Uniquement mêlée + Griffons.

#### Citadelles maudites

- **Niveau 20** : 2000 catapultes niveau 4. Uniquement mêlée + Griffons.
- **Niveau 25** : 7800 catapultes niveau 5. Uniquement mêlée + Griffons.

*Les unités entre parenthèses () sont optionnelles.*
