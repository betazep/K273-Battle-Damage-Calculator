<div class="book-section">
    <h4>Comment vérifier la protection de l'entrepôt</h4>
    <ul>
        <li>Entrez dans votre ville.</li>
        <li>Appuyez sur votre bâtiment capital.</li>
        <li>Sélectionnez "statistics".</li>
        <li>Sélectionnez "resources".</li>
    </ul>
</div>
<p>Ensuite, appuyez sur chaque ressource pour voir la quantité protégée. Les ressources en dessous ne peuvent pas être volées.</p>
