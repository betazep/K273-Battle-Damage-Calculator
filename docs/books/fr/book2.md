Les points de richesse du clan proviennent des coffres ouverts dans le clan. Chaque coffre vaut un score qui augmente avec la rareté. Un coffre gagné est attribué à tous les membres, donc un clan plein et actif est crucial. C'est pourquoi Carter est si important !

Les supérieurs et leaders peuvent récupérer ces cadeaux toutes les 6,5 h ici :

- Menu du clan
- Wealth
