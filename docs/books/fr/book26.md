#### Conseil de collecte de ressources

À mesure que les territoires de clan s'agrandissent, il est parfois difficile de savoir si une ressource est "sauvage" ou sur un autre territoire de clan.

- À côté de la ressource, essayez d'ouvrir un portail. Si vous êtes sur un territoire de clan, vous serez averti.
- Si vous n'êtes pas averti, ouvrez un portail si la distance est grande, sinon marchez jusqu'à la ressource.
