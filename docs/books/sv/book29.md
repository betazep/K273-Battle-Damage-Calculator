#### Pocket Fort - del 1

Klanens huvudstad skyddas av dina fort. När forten förstörs får huvudstaden 20h sköld.

Skadade fort kan inte repareras på 40h.

#### Vad kan du göra?

Avancerad strategi i del 2.

Kort förklaring inför öppning.

- Håll ett fort obygd.
- Bygg temporärt och riv reservfortet vid bonus.
- Om forten förstörs byggs det obygda fortet för skydd.
