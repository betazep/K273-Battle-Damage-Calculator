#### Doomsday

**Offra din armé för XP och Valor (trupperna dör).**

- Aktivera bonusar i förväg.
- Om du inte kan skicka stora arméer, använd bara de starkaste.
- Om du kan bygga stora arméer, stacka trupper.
- Prioritera Carter.
- Använd Valor till Archeology och Guardsmen.

**Spara mercenaries (epic monster hunters) till dessa event.**
