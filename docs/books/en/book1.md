If you use higher tier catapults than what is listed, only half the amount needs to be sent.

#### Elven citadels

- **Level 10**: 100 tier 2 catapults. Only ranged + Gargoyles.
- **Level 15**: 320 tier 3 catapults. Only Mounted + Gargoyles, Boars and Phoenix.
- **Level 20**: 800 tier 4 catapults. Only Melee + Griffins.
- **Level 25**: 3200 tier 5 catapults. Only Melee + Griffins.
- **Level 30**: Boop &amp; pray, tier 6 catapults. Only Melee + Griffins.

#### Cursed citadels

- **Level 20**: 2000 tier 4 catapults. Only Melee + Griffins.
- **Level 25**: 7800 tier 5 catapults. Only Melee + Griffins.

*Units listed in brackets () are optional.*
