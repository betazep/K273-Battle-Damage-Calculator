#### Событие Summon Mastery

Это событие дает шанс получить части пазла для конкретного капитана. Elite scroll дает 10 очков, basic scroll — 1 очко.

Собирайте очки, чтобы подниматься по личным заданиям.

Капитан меняется каждый ивент. Советы:

- Сохраняйте все scrolls.
- Ждите события с полезным капитаном.
- Используйте scrolls по 10 для лучшего шанса на высокие части.
- Место призыва находится в вашем городе.

Отлично для частей Carter, Strorr, Aydae и др. Следите за Events!
