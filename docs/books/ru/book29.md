    <h4>Pocket Fort — часть 1</h4>
    <p>Столица клана защищена вашими фортами. Когда форты разрушены, столица получает 20-часовой щит.</p>
    <p>Поврежденные форты нельзя ремонтировать 40 часов. Это оставляет 20-часовое окно для атаки.</p>
</div>
<div class="book-section">
    <h4>Что можно сделать?</h4>
    <p>Есть продвинутая стратегия защиты. Полные инструкции будут в части 2.</p>
    <p>Это короткое объяснение для подготовки.</p>
</div>
<ul>
    <li>Держите один форт незастроенным.</li>
    <li>Чтобы активировать другой бонус, временно стройте и затем разрушайте запасной форт.</li>
    <li>Если форты разрушены, незастроенный форт строится для восстановления щита.</li>
</ul>
<p>Полные шаги будут в части 2. Сейчас время освободить или разблокировать запасной форт.</p>
