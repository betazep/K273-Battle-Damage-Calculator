#### Телепорт города

Как телепортировать город:

- Найдите место.
- Нажмите на клетку для телепорта.
- Выберите "city teleport".
- Если нет телепорта, купите за золото.

#### Куда телепортироваться?

- Для больших армий нужна клетка с бонусом "food production". Армии требуют пищи.
- Для серебряных городов лучший выбор — пустыня.

**Важно:** Держите город как можно ближе к "группе" клана.
