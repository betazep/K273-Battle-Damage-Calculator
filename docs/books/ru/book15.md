<div class="book-section">
    <h4>Телепорт города</h4>
    <p>Как телепортировать город:</p>
    <ul>
        <li>Найдите место.</li>
        <li>Нажмите на клетку для телепорта.</li>
        <li>Выберите "city teleport".</li>
        <li>Если нет телепорта, купите за золото.</li>
    </ul>
</div>
<div class="book-section">
    <h4>Куда телепортироваться?</h4>
    <ul>
        <li>Для больших армий нужна клетка с бонусом "food production". Армии требуют пищи.</li>
        <li>Для серебряных городов лучший выбор — пустыня.</li>
    </ul>
</div>
<p><strong>Важно:</strong> Держите город как можно ближе к "группе" клана.</p>
