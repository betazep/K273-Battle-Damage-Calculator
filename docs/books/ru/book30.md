Это событие специально для прокачки героя.

- Зарабатывайте XP и Valor.
- Можете повышать героя.
- Можно открыть больше дерева навыков на lv65.
- Получайте очки навыков для героя.

#### Событие с предупреждением

**Прокачка героя:** Будьте осторожны. Уровень героя должен быть около половины уровня лучшего капитана.

Слишком быстрая прокачка может усложнить задания и события.

Прокачивайтесь, но помните об этом.
