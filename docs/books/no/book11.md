#### Meny

Bonuses

På denne siden kan du se ulike bonuser og aktivere dem. Å bruke gull på 30-dagers bonuser er et smart valg! Du får også gratis aktiveringer fra belønninger.

#### Personlig

I denne delen kan du aktivere mange bonuser for egen bruk. Aktiver disse før du kjemper epic monsters.

#### Klan

I denne delen kan du se tilgjengelige klanbonuser og hvilke som er aktive.

#### Royal

I denne delen kan du se bonusene aktivert av kongen.

#### Premium

Dette koster ekte penger, men du kan bytte fordeler hvert 15. minutt. Flott for bygging og krypting!
