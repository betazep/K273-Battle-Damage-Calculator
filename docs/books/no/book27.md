Denne guiden er basert på min erfaring og kunnskap som voksende leder.

- Sørg for at nye spillere er aktive og responsive. Aktive spillere som åpner kister driver velstand. Vett dem tidlig.
- Utdann spillere om viktigheten av å prioritere Carter med leveling og research.
- Sett ukentlige kryptmål for medlemmer og gjennomgå dem. Vi nådde 30m velstand med 45 spillere med 21 ukentlige kryptmål.
- Driv Tin-man deltagelse. Dette eventet gjør underverker for velstandsscoren. Følg med på deltagelse.
- Utdann medlemmer om fordelene med golden pass. Oppmuntre, men tving aldri til å bruke ekte penger. Passet øker Tar-inntekten.
- Utdann nye medlemmer ved forespørsel om Wellsprings-fordeler.
- Utdann klanen om fordelene og belønningene ved krypting.

**1 kiste x (antall medlemmer) x aktivitet = velstand**
