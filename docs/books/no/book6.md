#### Doomsday

**Ofre hæren din for XP og Valor (troppene dine vil dø).**

- Aktiver bonuser på forhånd.
- Hvis du ikke kan sende store hærer, bruk kun de sterkeste troppene.
- Hvis du kan bygge store hærer, stack troppene.
- Prioriter å level Carter.
- Bruk Valor til å forske på Archeology og Guardsmen.

**Spar mercenaries (epic monster hunters) til disse eventene.**
