<div class="book-section">
    <h4>Consiglio raccolta risorse</h4>
    <p>Man mano che i territori del clan crescono, è difficile sapere se una risorsa è "selvaggia" o in un altro territorio.</p>
</div>
<ul>
    <li>Vicino alla risorsa, prova ad aprire un portale. Se sei in territorio di un altro clan, verrai avvisato.</li>
    <li>Se non vieni avvisato, apri un portale se la distanza è grande o marcia senza aprirlo.</li>
</ul>
