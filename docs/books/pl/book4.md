#### Jak zbudować swój vault

- Wykonuj codzienne zadania.
- Przyzwij vault w zakładce Events.
- Zwołaj raid.
- Usuń graczy, którzy nie stosują się do zasad.
- Tylko pełne raid'y 5 graczy albo anuluj.

#### Strategia rajdu i kolory vault

- Zielony: minimum G2.
- Niebieski: minimum G3.
- Fioletowy: minimum G4.
- Złoty: minimum G5.
- Czerwony: minimum G6.
- Srebrny: minimum G7.

Nie dołączaj do rajdu, jeśli jesteś poniżej minimalnego poziomu Guardsmen. Jeśli inni dołączą poniżej, usuń ich.

#### Strategia wojsk

- Tylko Fighting Captains.
- Vault poziom 1-19: użyj Mounted i Range.
- Vault poziom 20-29: użyj Mounted.
- Vault poziom 30-39: użyj Spearmen i Archers.
