Si usas catapultas de un nivel más alto que el indicado, solo necesitas la mitad de la cantidad.

#### Ciudadelas élficas

- **Nivel 10**: 100 catapultas nivel 2. Solo rango + Gárgolas.
- **Nivel 15**: 320 catapultas nivel 3. Solo montados + Gárgolas, Jabalíes y Fénix.
- **Nivel 20**: 800 catapultas nivel 4. Solo cuerpo a cuerpo + Grifos.
- **Nivel 25**: 3200 catapultas nivel 5. Solo cuerpo a cuerpo + Grifos.
- **Nivel 30**: Boop & pray, catapultas nivel 6. Solo cuerpo a cuerpo + Grifos.

#### Ciudadelas malditas

- **Nivel 20**: 2000 catapultas nivel 4. Solo cuerpo a cuerpo + Grifos.
- **Nivel 25**: 7800 catapultas nivel 5. Solo cuerpo a cuerpo + Grifos.

*Las unidades entre paréntesis () son opcionales.*
