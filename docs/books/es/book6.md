#### Doomsday

**Suicida tu ejército para ganar XP y Valor (tus tropas morirán).**

- Activa bonos previamente.
- Si no puedes enviar grandes ejércitos, usa solo tropas más fuertes.
- Si puedes construir grandes ejércitos, apílalos.
- Prioriza subir a Carter.
- Usa Valor para investigar Archeology y Guardsmen.

**Guarda mercenaries (cazadores de monstruos épicos) para estos eventos.**
