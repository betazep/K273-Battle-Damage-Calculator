Pokud používáte katapulty vyšší úrovně než je uvedeno, stačí poslat pouze polovinu množství.

- **Úroveň 10**: 100 katapultů 2. úrovně. Pouze dálkové + gargoylové.
- **Úroveň 15**: 320 katapultů 3. úrovně. Pouze jezdci + gargoylové, kanci a fénixové.
- **Úroveň 20**: 800 katapultů 4. úrovně. Pouze zblízka + grifoni.
- **Úroveň 25**: 3200 katapultů 5. úrovně. Pouze zblízka + grifoni.
- **Úroveň 30**: Ťukni a modli se, katapulty 6. úrovně. Pouze zblízka + grifoni.

#### Prokleté citadely

- **Úroveň 20**: 2000 katapultů 4. úrovně. Pouze zblízka + grifoni.
