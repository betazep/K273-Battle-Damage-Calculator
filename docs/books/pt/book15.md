#### Teleporte de cidade

Como teleportar sua cidade:

- Encontre um local.
- Pressione o tile para onde deseja teleportar.
- Selecione "city teleport".
- Se você não tiver teleport, terá que comprar com moedas de ouro.

#### Para onde teleportar?

- Para exércitos maiores, você precisa de um tile com bônus de "food production". Exércitos precisam de comida. Quanto maior a produção, maior o exército.
- Para cidades baseadas em prata, o deserto é o melhor lugar.

**Importante:** Mantenha sua cidade o mais próxima possível do "grupo" do clã. Isso facilita reforços, caravanas, etc.
