Os pontos de riqueza do clã vêm dos baús abertos no clã. Cada baú vale uma pontuação que aumenta com a raridade. Um baú ganho é atribuído a todos os membros, então um clã cheio e ativo é crucial. É por isso que Carter é tão importante!

Superiores e líderes podem resgatar esses presentes a cada 6,5h aqui:

- Menu do clã
- Wealth
