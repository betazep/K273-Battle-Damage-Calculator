#### Doomsday

**Sacrifique seu exército para ganhar XP e Valor (suas tropas vão morrer).**

- Ative bônus antes.
- Se não puder enviar grandes exércitos, use apenas as tropas mais fortes.
- Se puder construir grandes exércitos, empilhe tropas.
- Priorize Carter.
- Use Valor para pesquisar Archeology e Guardsmen.

**Guarde seus mercenaries (caçadores de monstros épicos) para esses eventos.**
