## Règles du chat du Sénat - Élection du Roi

### Objectif
Le chat K273 King Election existe pour que les joueurs du Top 100 du tournoi puissent discuter et voter pour notre prochain Roi. Restons concentrés et respectueux.

### Administration du chat
Ce chat est administré par moi (Nakasa). Je surveillerai et appliquerai activement ces règles. Les joueurs qui ne les respectent pas seront retirés du chat du Sénat.

### Règles principales

#### 1. Rester dans le sujet
Les discussions doivent porter sur les nominations au Roi, les qualifications et le vote.
Les discours de campagne et les questions aux candidats sont les bienvenus.
Les conversations hors sujet doivent aller dans d'autres chats.

#### 2. Une nomination par message
Lorsque vous nommez quelqu'un, utilisez un format clair : "Je nomme [Nom du joueur] pour Roi".
Incluez 1 à 2 phrases expliquant pourquoi.
Évitez de nommer en spam la même personne de manière répétée.

#### 3. Débat respectueux uniquement
Soutenez les forces de votre candidat, ne dénigrez pas les autres.
La critique des qualifications est acceptable ; les attaques personnelles ne le sont pas.
Gardez un langage approprié pour tous les joueurs.

#### 4. Pas de spam de campagne
Limitez l'auto-promotion ou la promotion d'un candidat à 3 messages par heure.
Pas de cris en MAJUSCULES.
Pas de spam de copier-coller du même message.

#### 5. Processus de vote
Tous les votes officiels doivent être effectués via la page d'événement dans le jeu.
Ce chat est uniquement pour la discussion et le débat.
Ne publiez pas de messages "Je vote pour X" ici - allez sur la page d'événement.
Un vote par joueur - vous ne pouvez pas modifier votre vote sur la page d'événement une fois voté.

### Conséquences
Première infraction : Avertissement.
Deuxième infraction : Sourdine du chat pendant 24 heures.
Troisième infraction : Retiré du chat du Sénat pour le reste de l'élection.

Je me réserve le droit de retirer immédiatement tout joueur en cas d'infractions graves (harcèlement, menaces ou perturbation extrême).

### Rappel
Nous sommes le Top 100. Agissons comme des leaders et donnons l'exemple pour le royaume.
